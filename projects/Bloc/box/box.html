BlocBox

The alternative to Google Drive and Drop Box

Theme

BlocBox was the project developed during the design foundation of the Bloc program. In other words, it was the first project and the project that the program held my hand through. Having been trained as a contemporary architect, the overall layout and font decesions were completed rather fast. The one thing that I had a harder time to settle on was the color pallette. I eventually chose to focus on a blue to emphasize the professional aspect of the website and use a green to serve as a soft contrast here and there without brining in too much warm color.

Process

BlocBox being the first semi-real project, it was the project that I did the most in depth research and detailed documentation of each steps. After having done BlocBox I learned that a lot of the process can be combined together or done in a different method to make more efficent use of time.
